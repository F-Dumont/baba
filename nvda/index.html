🎮 Q1：英伟达到底是做什么的？

一句话回答：英伟达是一家做 GPU（图形处理器）的公司，但现在它的 GPU 不只是用来打游戏，而是被大量用于人工智能（AI）计算——就像盖房子需要砖头，训练AI需要大量算力，而英伟达的GPU就是那个最核心的"砖头"。
💻 以前（游戏时代）
英伟达 = 游戏显卡公司
GPU = 让游戏画面更精美的芯片
主要客户 = 游戏玩家
⚙ 现在（AI时代）
英伟达 = AI算力基础设施
GPU = 训练AI模型的"超级大脑"
主要客户 = 微软/谷歌/亚马逊等云厂商
类比理解：英伟达就像"卖铲子"的公司——19世纪加州淘金热时，真正发财的不是淘金者，而是卖铲子的商人。现在AI热，英伟达就是那个"铲子公司"，而且它不只卖铲子，还设计整个矿场系统。

📈 Q3：英伟达赚了多少？（财务术语解读）

🔑 重要概念：GAAP vs Non-GAAP
GAAP = 美国通用会计准则（法定规则）下的数字；Non-GAAP = 剔除股权激励等"一次性项目"后的调整数字。
黄仁勋在公开场合喜欢说Non-GAAP数字，因为它通常"更好看"。投资者要看两个。
指标数字大白话解读
营收 FY26全年$2,159亿相当于一个中等国家一年的GDP，3年增长14倍
GAAP净利润$1,201亿每赚100元，有55元是净利润（净利率55.6%）
Non-GAAP净利润$1,170亿调整后净利润，两者差距仅$30亿（股权激励影响小）
GAAP毛利率71.1%每卖100元的东西，成本只有29元——极高！但FY26比FY25下降了3.9个百分点
GAAP EPS$4.90每股赚$4.90，比FY25的$2.94增长67%
⚠ 需要警惕：毛利率从FY25的75.0%下降到FY26的71.1%（-3.9个百分点）。这意味着竞争开始加剧，或者成本（台积电代工费）开始上升。

🛡 Q5：英伟达面临哪些风险？

1. 美国芯片出口管制
美国政府限制英伟达向中国出口最先进GPU（H100/H200等)，直接影响约15%的营收。2026-04-14彭博报道：美国商务部BIS审批迟滞，进一步影响芯片出口。
影响：中国是全球第二大AI市场，不能卖给中国=丢失大量潜在收入。
2. 大客户自研芯片
谷歌有TPU、亚马逊有Trainium、微软有Maia——这些大客户都在自己研发AI芯片，理论上会减少对英伟达的依赖。
影响：英伟达最大客户（超大规模云厂商）正在"自建厨房"，可能减少采购。
3. 竞争加剧导致毛利率下降
AMD的MI300X性能大幅提升；英伟达FY26毛利率已从75%降到71%，说明竞争开始压价。
4. AI投资热潮可能降温
如果AI落地进展慢于预期，企业可能会削减AI基础设施投资。

力量二：新进入者威胁（3/10 — 低威胁）

CUDA生态是英伟达最宽的护城河。20年的积累让它成为AI编程的事实标准——全球数百万开发者只会用CUDA，新进入者需要说服他们重新学习，成本极高。

唯一突破口：如果某个新进入者能在软件生态上实现对CUDA的兼容性突破（例如AMD ROCm生态加速成熟），护城河会被部分侵蚀。目前看这需要3-5年以上。

力量四+五：供应商与客户议价能力（4/10 — 中低）

供应商（台积电）— 议价能力 4/10

台积电是全球唯一能生产3nm/5nm先进制程的代工厂，英伟达GPU全部依赖台积电制造。

分析：台积电议价能力强，但NVIDIA是其最大客户之一（占台积电收入~15-20%），双方是互相依赖的对等关系。

风险点：台积电若遭遇自然灾害/地缘冲突（台湾问题），将是英伟达供应链最大的黑天鹅。

客户（超大规模云厂商）— 议价能力 4/10

英伟达最大客户（AWS/Azure/GCP/Oracle）合计贡献约60%营收，单个客户议价能力确实强。

分析：大客户已开始自研芯片（谷歌TPU、亚马逊Trainium、微软Maia），议价能力在边际上升。

但NVIDIA有反制手段：CUDA生态锁定+NVLink网络整合让客户迁移成本极高。
